Streamer Name Integrator
一个功能强大的主播名字整合工具，支持多种分隔符识别、保持原始排版、紧凑显示和丰富的编辑功能。

功能特性
🎯 核心功能
智能名字整合：合并两个主播名单，自动去重，保留原始格式

多种分隔符支持：支持顿号（、）、中文逗号（，）、英文逗号（,）、空格和换行符

保持原始排版：整合时完全保持输入文本的原始格式（换行、空格、分隔符类型）

双向同步：可编辑输出框与交互列表之间可双向同步数据

🛠️ 编辑功能
交互式删除：点击删除按钮从列表中移除特定主播

删除恢复：支持恢复已删除的主播名字

一键撤回：撤回最近一次删除操作

历史记录：完整记录所有操作，支持导出历史

📊 数据显示
紧凑显示模式：节省空间，每行显示多个主播名字

网格显示模式：整齐的网格布局，便于浏览

列表显示模式：传统的垂直列表显示

实时统计：实时显示名单人数、重复人数、整合后人数等统计信息

📁 导入导出
文本导出：导出整合结果为TXT文件

文本导入：从TXT文件导入主播名单

历史导出：导出操作历史记录

一键复制：快速复制整合结果到剪贴板

使用方法
1. 输入主播名单
在"名单1"和"名单2"文本框中输入主播名字

支持使用顿号、逗号、空格或换行分隔名字

实时显示每个名单的人数统计

2. 整合名单
点击"整合名单"按钮合并两个名单

自动去重，重复名字只保留一个

保持名单1的原始排版格式，新增名字追加到尾部

3. 编辑和管理
在"整合输出文本"框中直接编辑最终名单

使用"交互式结果列表"的删除按钮管理主播

切换三种显示模式（紧凑/网格/列表）

点击已删除的名字可快速恢复

4. 导入导出
使用"导出整合文本"保存结果

使用"导入整合文本"加载已有名单

使用"复制整合文本"快速分享结果

主播D 主播E 主播F
